# Network Scanner

## Beschrijving

dit is een project voor het ontwikkelen van een netwerkscanner die apparaten binnen een lokaal netwerk kan detecteren en identificeren. Het project omvat zowel hardware- als softwarecomponenten en is bedoeld voor educatieve doeleinden binnen VIVES.

dit word uitgevoerd volgens de agile methodologie, met regelmatige sprints en feedback loops om continue verbetering te waarborgen.
