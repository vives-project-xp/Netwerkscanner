# Documentatie map

Deze map bevat alle projectdocumentatie voor de Netwerkscanner. Het doel is om de keuzes, ontwerpen en instructies die niet in de broncode thuishoren, centraal te bewaren zodat het project makkelijk te begrijpen, te onderhouden en te presenteren is.

Wat je hier kan verwachten:

- Ontwerp- en brainstormbestanden die de richting van het project tonen.
- Technische uitleg en installatiestappen, bijvoorbeeld voor hardware of een specifieke setup.
- Aanvullende notities, beslissingen en achtergronddocumentatie die tijdens het project werden gemaakt.

Gebruik deze map om documentatie up-to-date te houden en ervoor te zorgen dat nieuwe teamleden of begeleiders snel inzicht krijgen in de aanpak en werking van het project.
