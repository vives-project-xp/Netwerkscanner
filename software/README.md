## Software map

Deze map bevat alle broncode en softwaregerelateerde onderdelen van de Netwerkscanner. Het doel is om de implementatie gestructureerd te bewaren en duidelijk te maken hoe de software is opgebouwd en gebruikt kan worden.

Wat je hier kan verwachten:

- Broncodebestanden en modules van de scanner.
- Configuratiebestanden en voorbeeldinstellingen.
- Scripts voor installatie, build of deployment.
- Eventuele testbestanden en hulpmiddelen voor debugging.

Gebruik deze map om de software logisch te organiseren en de ontwikkeling en het onderhoud te vergemakkelijken.
